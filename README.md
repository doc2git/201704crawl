# 珠峰爬虫
## 初始化项目
```
npm init -y
```
## 安装依赖模块
```
npm install request iconv-lite cheerio debug async express mongoose ejs cron -S
```

## 如何安装布署
### 购买机器
### 连接服务器
```
ssh root@116.62.41.234
```

### 更新操作系统
```
apt-get update
```

### 安装 npm
```
apt-get install npm
```

### 安装node版本管理器 n
```
npm install n -g
n 7.5
```

## 安装mongodb
```
apt-get install mongodb
```

## 安装git
```
apt-get install git
```
